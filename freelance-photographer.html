FAQ

Questions from photographers specifically

Can I track gallery delivery deadlines and shooting schedules?

Yes. The Project Manager has date fields for shoot date, editing deadline, and gallery delivery date. You can filter by upcoming deliveries to see what's due this week and next. It's designed to handle multiple overlapping bookings, which is the norm during peak season.

Does it handle different license types (personal vs. commercial)?

Yes. The Contract Vault lets you note the license type and usage terms for each client contract. You can add a "license upgrade request" tag when clients ask to use images beyond their original scope — giving you a clean record of each license conversation and a way to invoice the difference.

Can I manage both weddings and commercial clients in the same system?

Yes — and it's one of the most common use cases. You create a project template for each shoot type (wedding, portrait, commercial, real estate), duplicate it per booking, and customize the deliverables. The Finance Dashboard lets you filter income and expenses by shoot type so you can see which category is most profitable.

I use HoneyBook (or another CRM). Why would I switch to Notion?

You don't have to switch entirely — some photographers use FreelanceOS as a central command center while keeping HoneyBook for client-facing communication like questionnaires and galleries. Others find FreelanceOS gives them more flexibility and costs less. Either way, it's $29 once vs. $19–$40/month for specialty tools, and Notion's flexibility means you can build exactly the system you want.
