Set Up Automatic Delivery (5-min setup)

Deploy a Cloudflare Worker that catches Stripe webhooks and sends emails automatically. No more manual steps.

1
Fill in template-config.js first — open it and paste your Notion share URLs for each product. Save the file and push to GitHub.
2
Create a Cloudflare account at cloudflare.com (free tier is sufficient). Then go to Workers & Pages and create a new Worker.
3
Set up a transactional email service — sign up for Resend.com (free up to 3,000 emails/month). Get your API key from the dashboard.
4
Deploy the webhook worker — the Worker listens for Stripe checkout.session.completed events, reads the product from the redirect URL's ?product= param (or metadata), looks up the Notion URL from config, and emails the buyer via Resend. Set STRIPE_WEBHOOK_SECRET and RESEND_API_KEY as Worker secrets.
5
Register the Stripe webhook — in Stripe Dashboard → Developers → Webhooks, add the Worker URL as an endpoint listening for checkout.session.completed. Copy the webhook signing secret into your Worker's environment variables.
6
Test with a $0 test purchase — make a test payment and confirm the email arrives. From that point on, delivery is fully automatic within seconds of purchase.
