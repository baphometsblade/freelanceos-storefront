Deep dive

Inside the Dashboard: Widget by Widget

Each widget is a purpose-built view linked to a live database. Here's what each one tracks and why it matters to your business.

Revenue Tracker

Pulls from the Invoice database using rollup properties. Totals paid invoices by month, calculates month-over-month growth, and shows a target completion percentage. The bar chart auto-updates as new invoices are marked paid — no formula editing required.

Client Health Board

Linked to the Client CRM database. Filters clients by status (Active / In Review / Idle / At Risk) and surfaces last-contact date so you can see immediately which relationships need a check-in. Prevents client churn from slipping through the cracks.

Project Pipeline

A filtered linked view of the Project Tracker database, showing only projects with a status of In Progress, Awaiting Feedback, or Blocked. Sorted by deadline so the most urgent work surfaces first. Click any row to open the full project card with brief, files, and task list.

Upcoming Invoices Panel

Filters the Invoice database to show invoices due within the next 30 days or flagged as overdue. Colour-coded: green for paid, amber for upcoming, red for overdue. Provides an at-a-glance cash flow picture without opening a separate finance tool.

Tasks Due This Week

A dynamically filtered view of the Task database showing only items with due dates in the current week. Prioritised by deadline and client. Supports the checkbox-complete interaction so you can tick items off directly from the dashboard without navigating to the full task list.

Income vs Target Progress Bar

A formula property that divides current month income by your target (set once in a settings page) and displays the result as a percentage. At 100%, the bar fills fully and a note prompts you to set a new target. One of the most motivating widgets in the system.
