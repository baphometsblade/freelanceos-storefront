Frequently Asked Questions

Can I manage multiple rental properties in this Notion template?
Yes. The Property Database supports any number of properties. Each property record includes address, type, number of units, purchase price, current value, mortgage info, and monthly rent totals. You can filter, sort, and view all properties in a portfolio summary view.
Does this template track maintenance requests?
Yes. The Maintenance Request Log tracks every request with fields for date submitted, property, unit, issue type, priority level, vendor assigned, current status (open/in progress/completed), and cost. You can filter open requests by property and prioritize urgent repairs.
Can I track lease expiration dates and renewals?
Yes. The Lease Tracker includes lease start date, end date, renewal status, security deposit amount, and lease document link. You can filter for leases expiring in the next 60 or 90 days so you're never caught off guard by an upcoming expiration.
Does this template include a rental income tracker?
Yes. The Rental Income Dashboard tracks monthly rent collected per property, expenses, net operating income (NOI), occupancy rate, and cash flow. You can see performance by property and compare month-over-month trends across your entire portfolio.
Is this suitable for professional property management companies or self-managing landlords?
Both. Self-managing landlords with 2-20 units find this template perfect for keeping everything organized without expensive property management software. Small property management companies with 50-100 units use it as a supplement to their property management platform for operational notes, vendor management, and portfolio reviews.
