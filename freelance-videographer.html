FAQ

Questions from videographers specifically

Can I customize the production stages to match my actual workflow?

Yes — Notion is fully flexible and FreelanceOS Pro is built to be customized. The default project stages work for most video workflows (pre-production, shoot, rough cut, revisions, color, audio, delivery), but you can rename or add stages to match exactly how you work. Takes about 5 minutes to set up once.

Does it handle both wedding videography and corporate/commercial work?

Yes. The system doesn't care about project type — you label it yourself. Most videographers create a project template per package type (wedding, corporate, social content), then duplicate it for each new booking. The invoice generator handles different payment structures (deposits, milestone billing, flat fee) with equal ease.

Can I track subcontractor costs like second shooters or audio engineers?

Yes. The Finance Dashboard lets you log expenses per project, including subcontractor payments. This gives you a true per-project profit margin — not just gross revenue but net after your cost of production.

Does this work if I also do video editing for other creators remotely?

Absolutely. Many buyers use it for a mix of production and remote editing work. Each project type gets its own template, and the CRM separates local production clients from remote editing clients. You can also track the CreatorHQ template for managing your own content creation alongside client work.
