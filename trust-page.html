The github.io question

You noticed this storefront runs on GitHub Pages. Here's why that's fine — and what actually protects you.

⚠ "Is a GitHub Pages site safe to buy from?"

It's a fair question. GitHub Pages is a free static hosting service — meaning this storefront doesn't have a database, a login system, or any infrastructure to get hacked. It's literally just HTML files.

GitHub Pages is used by thousands of legitimate small businesses and indie developers for exactly this reason: it's reliable, fast, and free to operate. You can see the source code is clean — no tracking scripts, no data collection.

But here's the more important point: you never give your payment details to this site. Every purchase goes through Stripe or Payhip — two of the most trusted payment processors in the world.
