Which Type of Student Are You?

Different student contexts require different Notion setups. Before building, identify your primary context — your planner should serve your actual academic life, not a generic student archetype.

🏫
Undergraduate
4–6 courses per semester, mix of lectures and labs, papers and exams, campus life and extracurriculars. Needs: assignment tracker, class schedule, grade calculator, and social/activity balance. Core challenge: preventing procrastination across too many simultaneous deadlines.
📚
Graduate Student
Fewer courses, more research, significant independent work, thesis or dissertation, often teaching or working as RA. Needs: research project tracker, literature review database, writing progress tracker, and deadline calendar. Core challenge: managing long-horizon projects with no external structure.
💻
Bootcamp Student
Intensive pace, project-based learning, portfolio building, job search simultaneous with studying. Needs: daily coding log, project portfolio tracker, skills learned database, job application pipeline. Core challenge: learning, building, and job-hunting simultaneously at high intensity.

The core databases in this guide apply to all three types. The emphasis and additional views differ — notes on those distinctions appear throughout.

Assignment Tracker Database

The assignment tracker is the operational core of your student planner. Everything else in Notion connects to it. Build it first and get comfortable with it before adding other databases.

The Assignment Database Properties

Property Type How to Use It
Assignment Name Title Be specific: "Essay 2 — Modernism in 20th Century Art" not "Essay." You'll thank yourself during exam week.
Course Relation → Courses DB Links each assignment to its course for grade rollup and schedule context.
Type Select Essay / Homework / Project / Exam / Quiz / Lab / Presentation / Reading
Due Date Date Always include time if there's a specific deadline. "11:59 PM" vs "in class" makes a difference.
Status Select To Do / In Progress / Submitted / Graded. Move through these as you work.
Priority Select High / Medium / Low. Based on grade weight AND proximity to deadline, not just deadline alone.
Grade % Number Enter your score once graded. Powers your course grade rollup.
Weight % Number How much of your final grade is this assignment? From the syllabus. Critical for grade calculations.
Notes Text Requirements, rubric points, professor instructions, peer feedback. Capture everything assignment-related here.

The Three Essential Assignment Views

View 1: Today & This Week (your default). Filter: Due Date is on or before 7 days from now AND Status != Graded. Sort by Due Date. This is your working view — the assignments in your immediate horizon that need attention.

View 2: Board by Status. Group by Status. All assignments across all courses, organized by where they are in progress. Moving a card from "To Do" to "In Progress" feels satisfying in a way that a task list doesn't — use this view to start work sessions.

View 3: Calendar by Due Date. See the entire semester spread across a calendar. This is the view that reveals crunch weeks months in advance — when you see four major deadlines within five days, you can start redistributing your work early rather than panicking the week before.

Semester Setup Tip

On day one of each semester, spend 45 minutes loading every assignment from every syllabus into your Notion database. All due dates, all assignment types, all weights. This one-time investment means you'll never miss a deadline due to forgetting it existed — and you'll always have the full-semester view when planning your week.

Class Schedule Setup

Your class schedule is more than a calendar — it's the second database in your student planner, linked to your assignments and notes. Create a Courses database with these properties:

  • Course Name (Title): Full name and course code: "PSYC 301 — Cognitive Psychology"
  • Professor (Text): Name, email, and office hours for quick reference
  • Meeting Times (Text): Days and times: "MWF 10:00–10:50am, Room 214"
  • Credits (Number): Credit hours for this course
  • Current Grade % (Rollup): Automatically calculated from linked assignments weighted by the Weight % property
  • Semester (Select): Fall 2026, Spring 2027, etc. Filter by semester to switch between active and archived courses
  • Syllabus Link (URL): Direct link to the course syllabus for reference
  • Color (Select): Assign each course a color. Use this for visual coding in calendar views

The Rollup for Current Grade is one of the most powerful features in this setup. Set it up by: (1) Adding a Rollup property in the Courses database, (2) Linking it to the Assignments relation, (3) Selecting "Grade %" as the property to roll up, (4) Choosing "Average" as the calculation. Every graded assignment you enter updates this number automatically.

Exam Countdown System

Exams are the highest-stakes assignments in any student planner. They deserve dedicated tracking beyond the standard assignment filter. Here's how to build an exam countdown system:

Step 1: Tag Exams
Mark All Exams with the Exam Type
When entering assignments from syllabi, set Type = Exam for all midterms, finals, and major tests. This enables exam-specific filtering across the entire semester database.
Step 2: Exam View
Create a Dedicated Exams View
Filter: Type = Exam AND Status = To Do. Sort by Due Date ascending. Add a formula property: dateBetween(prop('Due Date'), now(), 'days') to show how many days until each exam. Pin this view at the top of your planner dashboard — seeing "Finals in 18 days" every time you open Notion is exactly the low-level alarm that prevents last-minute cramming disasters.
Step 3: Study Plan Pages
Link Each Exam to a Study Plan
Open each exam entry and use the page body to build a study plan. List: topics covered, resources to review, practice problems to complete, and a day-by-day study schedule counting back from the exam date. This transforms an exam entry from a deadline into a project with a plan.

Study Session Tracker

Tracking study sessions serves two purposes: accountability (you see exactly how many hours you've actually put in) and calibration (you learn how long different types of work actually take you, so you can plan more accurately). Build a Study Sessions database:

  • Session Name (Title): Short description: "Cog Psych — Review Chapter 7 memory models"
  • Course (Relation → Courses): Which course was this session for?
  • Date (Date): When did you study?
  • Duration (Number): Hours studied (use decimals: 1.5 for 90 minutes)
  • Focus Score (Select, 1–5): How focused were you? 5 = deep focus, 1 = constantly distracted
  • Location (Select): Library / Home / Cafe / Campus / Other. Patterns emerge over time.
  • Assignment Linked (Relation → Assignments): Which assignment were you working toward?

In your Courses database, add a Rollup of Study Session Duration (Sum) to see total hours studied per course this semester. Compare this to your grade in each course — the correlation (or lack of it) is useful data about your study efficiency.

Proven Study Methods to Track

1
Pomodoro Technique
25 minutes of focused work, 5-minute break, repeat. After 4 rounds, take a 20-minute break. Track each session as a 25-minute entry. Research consistently shows this method beats long unstructured study sessions for retention.
2
Spaced Repetition
Review material at increasing intervals: 1 day, 3 days, 1 week, 2 weeks, 1 month. Track review dates in Notion using a "Next Review Date" property. This is the most evidence-backed method for long-term retention — critical for cumulative courses where early material appears on finals.
3
Active Recall
Instead of re-reading notes, close them and attempt to recall the material. Struggle-then-check is more effective than reading-then-checking. Log your active recall sessions separately from reading sessions in your Study Sessions database — they count more.
4
Teach-Back
Explain the material as if teaching someone else. The act of formulating an explanation reveals exactly where your understanding breaks down. Log these sessions as "Teach-back" in your session type and note which concepts needed clarification.

Grade Tracker

A grade tracker in Notion is more than logging scores — it's a real-time visibility system that tells you exactly where you stand in each course before the professor does. When you know your current grade is 78% with three weeks left, you can calculate exactly what score you need on the remaining assignments to hit your target grade. That clarity is motivating in a way that vague anxiety is not.

Grade Strategy

High-weight assignments (exams worth 30–40% of your grade) deserve disproportionate study time. If a homework set is worth 5% and an exam is worth 35%, an extra hour on exam prep is 7× more valuable than an extra hour on homework. Your Notion tracker makes this arithmetic visible and automatic.

Note-Taking System

Notion's combination of rich text, databases, and linking makes it an excellent note-taking system that beats standalone apps for students who are already using Notion for planning. The key is building notes that connect to your other databases.

The Notes Database

Create a Notes database (separate from the page body of your assignments) with these properties:

  • Note Title (Title): Short and searchable: "Memory Models — Atkinson-Shiffrin"
  • Course (Relation → Courses): Which course is this note from?
  • Lecture Date (Date): When was this lecture or reading?
  • Topics / Tags (Multi-select): Key concepts for cross-note search: "memory," "encoding," "retrieval"
  • Source (Select): Lecture / Textbook / Paper / Video / Other
  • Quality (Select): Complete / Needs Review / Incomplete. Flag notes you know are missing content.

The note content lives in the page body using these Notion blocks: H2 for major topics, H3 for sub-topics, Toggle blocks for details you want to collapse during review, Callout blocks for key definitions and formulas, and tables for comparison information.

Linking Notes to Assignments and Exams

Add a Relation property in your Notes database that links to Assignments. Before each exam, link all relevant notes to the exam entry — now the exam page becomes a study hub with all source material one click away. During exam prep, you see exactly which notes cover each topic the exam will test.

Managing Freelance Work Alongside Studies

Many students freelance — design students take on logo projects, writing students do content work, CS students build apps or do development contracts. Managing freelance work and academic work in the same Notion workspace creates clarity about where your time is actually going and prevents the common student-freelancer problem: over-committing to one and under-delivering on the other.

The practical approach: keep your student planner and your freelance business system in separate top-level pages in Notion (or separate Notion workspaces), but create a shared "This Week" planning page that pulls commitments from both. Each Sunday, your weekly planning session covers both: what do I need to deliver for clients this week? What academic deadlines are coming? Time-blocking both on your calendar creates an honest picture of your actual capacity before you commit to new work.

Frequently Asked Questions

Is Notion good for students?
Yes — Notion is one of the best tools for students because it handles both note-taking AND organization in a single app. Unlike paper planners (great for planning, bad for searchable notes) or simple note apps (great for capturing, bad for organizing), Notion lets you build a class schedule, assignment tracker, grade tracker, study session log, and course notes database all in one interconnected workspace. Notion is free for students with a .edu email via the Education plan.
How do I set up a Notion student planner?
Set up a Notion student planner by creating these core databases: (1) Assignments database with fields for Course, Due Date, Status, Type, and Priority. (2) Classes/Courses database with schedule, professor, and credit info. (3) Grade Tracker linked to both. Then create a dashboard home page with linked views: Today's Assignments (due today), This Week (due in 7 days), and Upcoming Exams. Add a class schedule view and a study session log. Start simple and add databases as you discover needs.
What should a Notion assignment tracker include?
A Notion assignment tracker should include: Assignment Name (title), Course (relation to Courses database), Type (Homework/Essay/Project/Exam/Quiz/Lab), Due Date (date), Status (To Do/In Progress/Submitted/Graded), Priority (High/Medium/Low), Estimated Hours (number), Actual Hours (number), Grade Received (number), and Notes (text). The most important view is a Board grouped by Status, with a filter showing only current semester assignments.
How do I track grades in Notion?
Track grades in Notion by adding a Grade % property (number) to your Assignments database. Then in your Courses database, add a Rollup property that averages the Grade % across all linked assignments. Add a Weight property to each assignment (matching the syllabus weighting), and create a weighted average formula: sum(grade * weight) / sum(weight). This gives you a running GPA estimate for each course that updates automatically as you enter grades.
Can I use Notion for class notes?
Yes, Notion is excellent for class notes. Create a Notes database linked to your Courses database. Each note entry has: Course (relation), Date, Topic, and the note content as the page body. Tag notes with keywords for searchability. The real power: you can link a note entry to a specific Assignment, so when reviewing for an exam, you can see all notes relevant to each upcoming assignment in one view. Use /toggle blocks for condensing long notes into scannable summaries.
How do bootcamp students use Notion differently from university students?
Bootcamp students face a faster pace and more project-focused learning than university students. Their Notion setup should emphasize: a Projects database (portfolio pieces, not just assignments), a Skills Tracker (what you've learned, what you need to learn), a Job Search pipeline (companies, application status, interview prep notes), and a Daily Coding Log (what you built or learned each day). Grade tracking matters less; portfolio and job-readiness tracking matters more. Many bootcamp grads keep their Notion workspace as they transition into their first role.
How do I create an exam countdown in Notion?
Create an exam countdown in Notion by adding a select property to your Assignments database with an 'Exam' type option. Then create a filtered view called 'Upcoming Exams' showing only entries where Type = Exam and Status != Submitted, sorted by Due Date ascending. Add a formula property: dateBetween(prop('Due Date'), now(), 'days') formatted as a number to show days remaining for each exam. Display this view prominently on your dashboard so exam deadlines are always visible.
