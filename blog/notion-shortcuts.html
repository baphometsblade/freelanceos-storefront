Why Notion Shortcuts Matter

The average Notion user takes 3–5 mouse clicks to do what a keyboard shortcut does in under half a second. That might sound trivial — but at 100 repetitive actions per day, the math adds up fast.

Notion's 3 Types of Shortcuts

Before the full reference, it helps to understand the three systems Notion uses for fast input:

  • Keyboard shortcuts — modifier key combinations (Cmd/Ctrl + key) that trigger formatting, navigation, or block actions. These work like shortcuts in any app.
  • Slash commands — type / anywhere in a page to open a block insertion menu. Filter results by typing (e.g., /h1 to jump straight to Heading 1).
  • @ commands — type @ to mention a person, page, date, or set a reminder. Inline, contextual, and fast.

Mac vs. Windows: On Mac, use Cmd where Windows users use Ctrl. The Option key on Mac maps to Alt on Windows. All shortcuts below use the Mac convention first, with Windows equivalent noted where different.

10 Power User Shortcuts Most People Don't Know

These are less obvious shortcuts that separate Notion beginners from power users. Most people discover them by accident — if ever.

Cmd + Shift + L
Toggle dark/light mode — switch between dark and light themes instantly without going into Settings.
Cmd + /
Change block type — with text selected or cursor in a block, opens a menu to convert the current block to any other type (H1, toggle, callout, etc.).
Esc
Exit text editing mode — press Esc while editing a block to deselect text and select the block itself. Then use arrow keys to navigate between blocks without clicking.
Cmd + Enter
Open page in full screen / side peek — when cursor is inside a linked page or database row, Cmd+Enter opens it. Also checks/unchecks to-do checkboxes.
Shift + Enter
Line break without new block — creates a soft line break inside the current block, keeping the content in one block instead of splitting it into two.
Tab / Shift + Tab
Indent / outdent — Tab indents a block (making it a child of the block above). Shift+Tab moves it back out. Essential for nested lists and toggle hierarchies.
Click block handle → drag
Drag to reorder or create columns — click the ⠿ drag handle (appears on hover at left of block), drag to reorder. Drag to the right side of another block to create a two-column layout.
Cmd + Click (link/page)
Open page in side peek — Cmd+Click on any page link or database row opens it in the right-side panel without losing your current page context.
Cmd + Shift + M
Add comment — opens the comment input on the selected block. Useful for collaborative review without disrupting the page content.
Cmd + Shift + P
Page history / version history — opens the page history panel where you can restore previous versions of any Notion page (Notion Plus and above).

Notion Shortcut Workflows by Use Case

Keyboard-First Writing Flow

For writers who want to stay in the keyboard the entire time:

  1. Open page: Cmd+P → type page name → Enter
  2. Start new section: # + Space (Heading 1)
  3. Format inline: Cmd+B, Cmd+I, Cmd+K
  4. Insert divider between sections: --- + Enter
  5. Add a callout: /callout + Enter
  6. Add a code block: ``` + Enter

Fast Database Entry Flow

For adding multiple rows to a database quickly:

  1. Open database and click the first empty row
  2. Fill in the title → press Tab to move to the next property
  3. Press Enter to save and move to a new row
  4. Press Esc when done entering data
  5. Press Cmd+Enter on any row to open it as a full page

Fast Page Navigation Flow

For bouncing between pages during a focused work session:

  1. Use Cmd+P (Quick Find) to jump to any page — it's fuzzy search, so partial titles work
  2. Use Cmd+[ and Cmd+] to navigate back and forward through recently visited pages
  3. Use Cmd+\ to hide the sidebar and maximize writing space
  4. Use Cmd+Click on linked pages to open them side-by-side without losing context

Frequently Asked Questions

What is the shortcut to create a new page in Notion?
Press Cmd+N on Mac or Ctrl+N on Windows to create a new page. You can also press Cmd+Shift+N to open a new Notion window. In the Notion desktop app, new pages are added to your sidebar automatically.
How do I bold text in Notion?
Select the text you want to bold and press Cmd+B on Mac or Ctrl+B on Windows/Linux. Alternatively, surround text with **double asterisks** — Notion auto-formats Markdown inline. You can also highlight text and click the Bold button in the floating toolbar that appears.
What are Notion slash commands?
Slash commands are triggered by typing / anywhere in a Notion page. They open an insertion menu letting you add headings, tables, databases, images, code blocks, callouts, and many other block types — without using your mouse. Start typing after the slash to filter the menu (e.g., /callout jumps straight to the callout option).
How do I duplicate a block in Notion?
Select a block by clicking it, then press Cmd+D (Mac) or Ctrl+D (Windows) to duplicate it. The copy appears immediately below the original. You can also click the six-dot drag handle on the left of any block and select Duplicate from the context menu.
What is the Notion search shortcut?
Press Cmd+P on Mac or Ctrl+P on Windows to open Notion Quick Find. This searches across your entire workspace with fuzzy matching — you don't need to type exact page titles. Results update as you type and include recently visited pages at the top.
