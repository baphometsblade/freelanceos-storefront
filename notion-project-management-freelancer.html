The 6 Databases Every Freelancer Needs in Notion

A complete freelance project management system in Notion is built on six relational databases. Here's what each one does:

👤

Clients Database

Every client you work with, past and present. Fields: name, company, contact info, status (active/inactive/lead), time zone, preferred communication channel, hourly rate or retainer amount, and notes. All projects, invoices, and meetings link back to this database.

📁

Projects Database

Every active and past project. Links to client, status (scoping/active/review/complete), start date, deadline, budget, and deliverables. Each project page contains briefs, meeting notes, file links, and a task list.

✅

Tasks Database

Every actionable task across all projects. Links to project and client. Filters let you see "all tasks for Client X," "all tasks due this week," or "all tasks currently in progress." This is your daily work queue.

💰

Invoices Database

Every invoice you send. Links to client and project. Tracks invoice date, amount, due date, and payment status (unpaid/paid/overdue). A rollup on the Clients database shows total lifetime value and outstanding balances.

📋

Proposals Database

Your pipeline of prospects and proposals. Status pipeline: lead → contacted → proposal sent → negotiating → won/lost. Win rate rollup shows what percentage of proposals become projects.

📞

Meetings Database

Structured notes for every client call. Links to client and project. Template prompts you for agenda, notes, decisions made, and next actions — so nothing falls through the cracks after a call.

Setting Up Your Freelance Project System: Step by Step

  1. Duplicate FreelanceOS Pro to your Notion workspace.
  2. Enter all current clients in the Clients database. Include their contact info, time zone, and rates.
  3. Create a project page for each active engagement. Link each project to the correct client.
  4. Enter all active deliverables as tasks in the Tasks database, linked to the relevant project.
  5. Log all outstanding invoices in the Invoices database so you immediately have visibility on cash flow.
  6. Set up your daily habit: Each morning, open the "Today's Tasks" view. Each Friday, run the weekly review. After each client call, fill in the meeting notes template.
