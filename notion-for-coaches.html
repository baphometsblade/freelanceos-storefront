Step-by-Step Setup

Setting Up Your Coaching OS

Follow this four-step blueprint to build a full coaching workspace in Notion in under two hours — no technical skills required.

1

Create Your Client Database

Start with a Clients table: Name, Status (Active / Paused / Alumni), Program, Start Date, Next Session, and a Notes relation. This becomes your master CRM. Every other database — sessions, goals, invoices — links back here. Add a Gallery view filtered to Active clients for your daily dashboard.

2

Build a Session Notes Template

Create a Sessions database linked to Clients. Add a repeating template with sections for Wins Since Last Session, What We Worked On, Key Insights, Action Items (checkboxes), and Next Session Date. Fill it in during or immediately after every call so nothing slips through the cracks.

3

Set Up Your Program Tracker

Build a Programs database with modules as sub-pages. Each module has a Description, Week Number, Resource Links, and an Exercises section. Relate programs to clients so you always know which client is on which module and how far along they are.

4

Connect Your Content Pipeline

Add a Content database with properties for Platform, Format (video / newsletter / post), Status (Idea / Draft / Scheduled / Published), and Publish Date. Filter by Status and Platform to manage your entire content operation from one board view and repurpose session insights into content ideas automatically.
