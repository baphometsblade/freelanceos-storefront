📋 How to Set Up Your Notion Templates

1

Download your ZIP file above

Click the download button and save the ZIP file to your computer. It contains CSV files and a full setup guide.

2

Open Notion and create a new page

Go to notion.so (free account works). Create a new page called “FreelanceOS” or “Business Hub.”

3

Import each CSV as a new database

In Notion: click + → Import → CSV. Import each file: clients.csv, projects.csv, invoices.csv, time-log.csv, proposals.csv. Each becomes a linked database.

4

Read the SETUP-GUIDE inside your ZIP

Open the Markdown file in any text editor. It explains how to connect the databases together with relations and set up your views.

5

You’re done — start adding your real data

Delete the sample rows and add your real clients, projects, and invoices. Your system is live.
